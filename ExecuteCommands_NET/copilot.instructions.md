# Copilot Instructions for SpeechRecognitionHelpers

- **Do not run automated tests for this project.**
- **Verification must be done by manual inspection of the log file (`app.log`).**
- When confirming fixes, always check the log output for expected results.
- If a user requests a test run, politely remind them that only manual log inspection is supported for this project.
- Document this rule in every session and update plans accordingly.
